# ODISS 데모 스토리 — 사람이 따라 하는 실험 가이드

김영수(72세, 남) 시나리오를 **실제 마이크·스피커·카메라**로 재현하기 위한 안내입니다.  
말할 문장은 가능한 한 **그대로** 사용하세요. 웨이크워드는 `오디스`, `오디스야`, `저기` 등이 인식됩니다.

## 2. 공통 사용법

1. **웨이크**: `오디스` (또는 `오디스야`) → 즉시 멘트(예: “네, 어르신 말씀해 주세요”) 들림  
2. **질문**: 멘트 끝난 뒤 **1초 쉬고** 질문 (너무 빨리 말하면 빈 STT로 무시될 수 있음)  
3. **끝맺음**: 문장 끝에서 **0.5초 정도 멈춤** (VAD가 발화 종료를 잡음)  
4. **촬영 트리거**: `약 찍어`, `약 가져왔어`, `사진 찍어`, `약 사진 좀 찍을게` 등  
5. **기대하지 말 것**: 스크립트의 “5, 4, 3…” 멘트는 **캡처 타이머(약 3초)** 로 대체될 수 있음. 의미만 같으면 OK.

## 스토리 #1 — 신규 사용자 등록

**목표**: 이름·성별·나이 저장 후 회상

| 순서 | 체험자 (말하기) | 기대 동작 | 통과 기준 |
|------|-----------------|-----------|-----------|
| 1 | `오디스` *(한 번만)* | 웨이크 → **서버** 인사 TTS | “일전에 대화”, “있나요” |
| 2 | *(잠깐 쉬고)* `김영수 남자고, 72살이야.` *(또는 `아니요` 후 동일)* | 프로필 저장 | “김영수”, “72”, “남” 포함 |
| 3 | *(같은 웨이크 세션, 다시 웨이크 없이)* `내가 누구인지 말해봐.` | 프로필 회상 | “김영수”, “72” 포함 |

**중요**

- `오디스`만 말하면 **로컬 “말씀해 주세요”만 나오던 구버전**이 아니라, 곧바로 **“일전에 대화한 적 있나요?”** 가 나와야 합니다.
- 바로 신원을 말해도 됩니다 (`김영수 남자고, 72살이야.`). **아니요**라고 하면 이름·성별·나이 등록으로 이어집니다.
- 등록·회상은 **한 번의 웨이크 안에서** 이어 말해도 됩니다 (최대 약 90초).
- STT가 `난 김영수고…`처럼 잡혀도 이름이 인식되도록 패치되어 있습니다.

**실패 시**: `reset_demo_speaker.py` 후 #1부터 재시도. `ODISS_SPEAKER_ID`가 매번 같아야 함.

## 스토리 #2-1 — OCR 및 약봉투 등록

**목표**: RTSP 촬영 → OCR → 서버 저장 → 식후 복약 안내 가능 상태

| 순서 | 체험자 | 기대 동작 | 통과 기준 |
|------|--------|-----------|-----------|
| 1 | `오디스` → `오디스. 내가 먹는 약 사진 좀 찍을게.` | 촬영 안내 + 카운트다운 TTS | “촬영”, “약” 류 멘트 후 셔터 구간 |
| 2 | 약봉투를 **카메라 앞에 고정** (글자 안 흔들리게) | Buffer → Timer → OCR | 10~60초 내 처리 멘트 |
| 3 | *(서버/에이전트가 인식 결과 읽어 주면)* `응, 저장해줘.` | 저장 확인 | “저장” 포함 응답 |
| 4 | `오디스` → `오디스. 나중에 밥 먹고 나면 뭐 먹어야 하는지 알려줘.` | 식후 안내 예약 멘트 | “식후”, “다시 말씀” 류 |

**진행자 메모**

- 촬영이 안 되면: `약 찍어` 만 따로 웨이크 후 발화  
- Cloud가 `ocr_request` 를내면 Jetson이 자동으로 촬영 상태로 전환됨  
- OCR 실패 시 #2-1-a 로 이동

**데모 팁**: 실시간 12시까지 기다리기 어렵으면 진행자가 “알림은 서버에 설정됨”만 확인하고, 복용 기록(4)까지 진행해도 됩니다.

## 5. 알려진 제한 (진행자용)

- TTS는 gTTS(네트워크) — 오프라인이면 합성 실패 가능  
- 신규 `speaker_id` 없이 `jetson_live`만 쓰면 매번 “처음 뵙는 분” 멘트가 나올 수 있음 → `demo_kimyoungsu` 권장  
- OCR 품질은 조명·초점에 크게 의존  
- 알림 실시간 재생은 에이전트가 **계속 실행 중**이어야 WebSocket `reminder` 수신 가능
